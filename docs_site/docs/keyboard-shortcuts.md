# Keyboard Shortcuts

## File Operations

| Action | Shortcut | Menu Location |
|--------|----------|---------------|
| Import MAT files | `Ctrl+L` | File → Import MAT files |
| Import XML file | `Ctrl+X` | File → Import XML file |
| Export XML file | `Ctrl+E` | File → Export XML file |
| Exit Application | `Ctrl+Q` | File → Exit |

## Navigation

| Action | Shortcut | Description |
|--------|----------|-------------|
| Switch between sensor tabs | Use mouse or click tabs | Navigate between loaded sensors |
| Zoom in/out | Mouse wheel | Zoom the signal plot |
| Pan plot | Click and drag (when zoomed) | Move around zoomed plot |

## Plot Interaction

| Action | Method | Description |
|--------|--------|-------------|
| Select burst region | Click and drag on plot | Manually select MVC burst |
| Clear selection | Click clear button (broom icon) | Remove selected burst |
| Switch sensor | Click radio button or tab | Change active sensor view |

## General

| Action | Shortcut | Menu Location |
|--------|----------|---------------|
| Open Documentation | - | Help → Documentation |
| About Dialog | - | Help → About |
| Request License | - | Help → Request License... |

## Tips for Efficient Workflow

### Using Shortcuts

1. **Import Files Quickly**: Use `Ctrl+L` to quickly load new files
2. **Save Frequently**: Use `Ctrl+E` to export results regularly
3. **Quick Exit**: Use `Ctrl+Q` to close the application

### Mouse Operations

- **Left Click + Drag**: Select burst regions
- **Mouse Wheel**: Zoom in/out on signal plot
- **Right Click**: (Context menu if available)

### Workflow Optimization

1. Load files with `Ctrl+L`
2. Use Energy Detection tool (button click)
3. Fine-tune selections with mouse
4. Process with button click
5. Export with `Ctrl+E`

---

## Customization

Currently, keyboard shortcuts are fixed. Future versions may allow customization.

For now, use the menu bar if you prefer mouse navigation over keyboard shortcuts.

---

## Platform-Specific Notes

### Windows

- All shortcuts work as listed
- `Ctrl` key is used (not `Cmd`)

### Linux

- All shortcuts work as listed
- `Ctrl` key is used

### Mac (if supported)

- `Ctrl` may need to be `Cmd` (check menu for actual shortcuts)
- Some shortcuts may differ

---

## Accessibility

If keyboard shortcuts don't work:

1. All functions are available via menu bar
2. All functions are available via button clicks
3. Contact support if you need alternative input methods
